# Love2D App

A simple Love2D (LÃ–VE) application template.

## Requirements

- [Love2D](https://love2d.org/) installed on your system

## Running the App

### Windows
Drag the project folder onto `love.exe`, or run:
```bash
love .
```

### Linux/Mac
```bash
love .
```

Or if love is not in your PATH:
```bash
/path/to/love .
```

## Controls

- `ESC` - Quit the application

## Features

- Basic game loop with animated circle
- FPS counter
- Configurable window settings
